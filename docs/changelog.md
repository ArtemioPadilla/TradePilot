# Changelog

All notable changes to TradePilot will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Web platform with Astro frontend
- Three theme system (Bloomberg, Modern, Dashboard)
- Portfolio dashboard with live metrics
- Firebase authentication integration
- Sphinx documentation with auto-generation

### Changed
- Refactored TPS simulator for improved performance
- Updated optimization module with configurable weights

## [1.0.0] - 2024-01-01

### Added
- Initial release of TradePilot
- Core backtesting engine (TPS)
- Live trading module (TPT)
- Market data retrieval (Yahoo Finance)
- Portfolio optimization (MSR, GMV, Equal Weight)
- Performance metrics calculation
- Built-in strategies:
  - Momentum
  - Mean Reversion
  - Smart Beta
- Alpaca broker integration
- Command-line interface
- Example notebooks

### Documentation
- Getting started guide
- API reference
- Strategy development guide
- Live trading guide
